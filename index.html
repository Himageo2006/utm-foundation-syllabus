Academic Listening & Speaking Skills

FSPE0022 · Language Academy, UTMJB

This subject trains you to communicate confidently in academic English. Your grade is built from four spoken/listening coursework tasks plus a final listening exam. Below: each assessment explained, and what graders look for.

SpeakingPresentationsDiscussionListening

🔍 Each task explained

20%Impromptu Speech▶

You're given a current-issue topic and must speak with little preparation. The goal is to organise your thoughts on the spot and deliver them confidently. Done twice (10% each).

Structure expected
  • Introduction — state your stance clearly.
  • Body — give your opinion with reasons.
  • Conclusion — wrap up your position.
What graders score (rubric)
  • Organisation — clear intro, body, conclusion.
  • Clarity of ideas — supported, with correct grammar & vocabulary.
  • Delivery — not scripted, conversational tone, gestures, eye contact.
  • Time management — best score for 2–3 minutes.
10%Oral Presentation▶

A prepared, formal informative speech (5–7 min) on a topic related to your field of study. You prepare, practise, then present.

Scored on
  • Speaking skills — language use, pronunciation, fluency, body language.
  • Content — strong introduction, two well-developed main points, clear conclusion, smooth transitions.
  • Visual aids — relevant, clear graphics; accurate, concise text.
  • Time management.
10%Group Discussion▶

In groups, you discuss a current-issue situation (10–15 min). It runs in two parts: a short individual presentation (Task A), then group interaction to reach a decision (Task B).

Scored on
  • Task fulfilment — logical, relevant ideas supported by facts/examples.
  • Language — correct expressions for inviting/giving opinions, agreeing & disagreeing, encouraging others.
  • Interactive ability — responding, taking turns, including others, concluding well.
  • Time management.

Example situation: "A neighbourhood park keeps getting vandalised — discuss the most effective way to prevent it."

20%+40%Listening Test & Final Exam▶

Audio-based assessments testing comprehension. Two coursework tests (10% each, Units 1–8) plus a 1-hour final exam (40%).

Skills tested
  • Identify the main idea and supporting details.
  • Make inferences and listen for opinions.
  • Identify cause and effect; predict what comes next.
  • Substitute/replace words with given words.
Question types
  • Match key words to definitions.
  • Multiple choice (circle the correct answer).
  • Fill in the blanks; information-transfer tasks.

Calculus

Mathematics · Semester 3

Calculus is the mathematics of change (differentiation) and accumulation (integration). It starts with limits, builds the two core operations, applies them to real problems, and ends with differential equations and numerical approximation. Every topic builds on the previous one.

📖 Open step-by-step Calculus lessons →

TOPIC 1Limits & Continuity▶

A limit describes the value a function approaches as the input approaches some point — the single most important idea underpinning all of calculus. Continuity means a function has no breaks, jumps or holes.

Key concepts
  • Evaluating limits by substitution, factoring, and rationalising.
  • One-sided limits (left and right).
  • Limits at infinity and infinite limits (asymptotes).
  • The three conditions for continuity at a point.
Key idea
limx→a f(x) = L  ·  continuous at a ⟺ limx→a f(x) = f(a)
TOPIC 2Differentiation▶

The derivative measures the instantaneous rate of change — the slope of the tangent to a curve. You'll learn the rules that let you differentiate almost any function.

Key concepts
  • Derivative from first principles (the limit definition).
  • Power, product, quotient and chain rules.
  • Derivatives of trigonometric, exponential and logarithmic functions.
  • Implicit differentiation and higher-order derivatives.
f′(x) = limh→0 [f(x+h) − f(x)] / h  ·  (uv)′ = u′v + uv′  ·  chain: dy/dx = dy/du · du/dx
TOPIC 3Applications of Differentiation▶

Using derivatives to understand and optimise real situations — where things are largest, smallest, increasing or decreasing.

Key concepts
  • Increasing/decreasing functions and critical points.
  • Maximum and minimum (optimisation problems).
  • Concavity, points of inflection, and the second-derivative test.
  • Related rates and curve sketching.
Max/min where f′(x)=0 ; concave up if f″(x)>0, concave down if f″(x)<0
TOPIC 4Integration▶

Integration is the reverse of differentiation and a way to add up infinitely many small pieces — used to find areas and totals.

Key concepts
  • Indefinite integrals (antiderivatives) and the constant of integration.
  • Definite integrals and the Fundamental Theorem of Calculus.
  • Integration by substitution.
  • Integration by parts.
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C  ·  ∫ₐᵇ f(x) dx = F(b) − F(a)
TOPIC 5Integration of Trigonometric Functions▶

Specialised techniques for integrating expressions involving sin, cos, tan and their powers.

Key concepts
  • Standard trig integrals (∫sin, ∫cos, ∫sec², …).
  • Using trig identities to simplify before integrating.
  • Powers of sine and cosine.
  • Trigonometric substitution.
∫ sin x dx = −cos x + C  ·  ∫ cos x dx = sin x + C  ·  ∫ sec²x dx = tan x + C
TOPIC 6Applications of Integration▶

Putting integration to work to compute geometric and physical quantities.

Key concepts
  • Area under a curve and area between two curves.
  • Volumes of revolution (disk and shell methods).
  • Average value of a function.
  • Applications to physics (distance, work).
Area = ∫ₐᵇ [f(x) − g(x)] dx  ·  Volume = π ∫ₐᵇ [f(x)]² dx
TOPIC 7Ordinary Differential Equations (ODEs)▶

Equations that relate a function to its derivatives — the standard way to model growth, decay, motion and many natural processes.

Key concepts
  • Order and degree of a differential equation.
  • Separable first-order equations.
  • Linear first-order equations (integrating factor).
  • General vs particular solutions; initial conditions.
Separable: ∫ 1/g(y) dy = ∫ f(x) dx  ·  e.g. dy/dx = ky ⇒ y = Ce^{kx}
TOPIC 8Numerical Methods▶

When an exact answer is impossible or hard, these methods give accurate approximations using arithmetic steps.

Key concepts
  • Root-finding: bisection method and Newton–Raphson method.
  • Numerical integration: trapezoidal rule and Simpson's rule.
  • Error and accuracy of approximations.
Newton–Raphson: xₙ₊₁ = xₙ − f(xₙ)/f′(xₙ)

Chemistry II

Physical & Organic Chemistry · Semester 3

Chemistry II is about energy, speed, balance and reactions: how much heat reactions exchange, how fast they happen, where they settle, how acids and bases behave — then an introduction to carbon-based (organic) chemistry.

CHAPTER 1Thermochemistry▶

The study of heat (energy) changes that accompany chemical reactions.

Key concepts
  • System vs surroundings; exothermic vs endothermic reactions.
  • Enthalpy change (ΔH) and enthalpy diagrams.
  • Heat of reaction, formation, combustion and neutralisation.
  • Hess's Law and calorimetry calculations.
q = mcΔT  ·  ΔHreaction = ΣΔHproducts − ΣΔHreactants
CHAPTER 2Chemical Kinetics▶

How fast reactions go and what controls their speed.

Key concepts
  • Rate of reaction and how to measure it.
  • Factors affecting rate: concentration, temperature, surface area, catalyst.
  • Rate law, rate constant and order of reaction.
  • Collision theory and activation energy.
Rate = k[A]ᵐ[B]ⁿ  ·  higher temperature ⇒ more successful collisions
CHAPTER 3Chemical Equilibrium▶

Reversible reactions reach a dynamic balance where forward and reverse rates are equal.

Key concepts
  • Reversible reactions and dynamic equilibrium.
  • Equilibrium constant Kc (and Kp).
  • Le Chatelier's principle: effect of concentration, pressure, temperature.
  • Industrial applications (e.g. the Haber process).
Kc = [products]/[reactants] (each raised to its coefficient)
CHAPTER 4Acids & Bases▶

The behaviour of acids and bases in solution, and how we measure acidity.

Key concepts
  • Arrhenius and Brønsted–Lowry definitions.
  • Strong vs weak acids/bases; the pH scale.
  • Ionisation, Ka / Kb, and the ionic product of water.
  • Neutralisation, titration and buffer solutions.
pH = −log[H⁺]  ·  Kw = [H⁺][OH⁻] = 1×10⁻¹⁴ at 25 °C
CHAPTER 6Organic Chemistry▶

The chemistry of carbon compounds — the basis of fuels, plastics and life itself.

Key concepts
  • Hydrocarbons: alkanes, alkenes, alkynes.
  • Functional groups (alcohols, carboxylic acids, etc.) and homologous series.
  • IUPAC naming (nomenclature) and isomerism.
  • Typical reactions: substitution, addition, combustion.

Physics II

Electricity, Magnetism, Optics & Modern Physics

Physics II moves through four themes: electricity (charge & circuits), magnetism, optics (light & waves), and modern physics (quantum & nuclear). The early electricity chapters are the foundation for everything later.

⚡ Electricity🧲 Magnetism🔦 Optics⚛️ Modern Physics
CH 1Electrostatics — Charge & Electric Field▶

Stationary electric charges and the forces and fields they create.

Key concepts
  • Electric charge, conductors and insulators.
  • Coulomb's law of electrostatic force.
  • Electric field and field lines.
  • Electric potential and potential energy.
F = k q₁q₂ / r²  ·  E = F/q
CH 2Capacitors▶

Devices that store electric charge and energy.

Key concepts
  • Capacitance and the parallel-plate capacitor.
  • Effect of dielectrics.
  • Capacitors in series and parallel.
  • Energy stored in a capacitor.
C = Q/V  ·  E = ½CV²
CH 3Electric Current & Resistance▶

The flow of charge and what opposes it.

Key concepts
  • Current, drift velocity and Ohm's law.
  • Resistance, resistivity and factors affecting it.
  • Electrical power and energy.
V = IR  ·  P = VI = I²R
CH 4DC Circuits▶

Analysing direct-current circuits with multiple components.

Key concepts
  • Resistors in series and parallel.
  • EMF and internal resistance.
  • Kirchhoff's current and voltage laws.
Series: R = R₁+R₂+…  ·  Parallel: 1/R = 1/R₁+1/R₂+…
CH 5Magnetism▶

Magnetic fields and the forces they exert on moving charges and currents.

Key concepts
  • Magnetic field and field lines.
  • Force on a moving charge and on a current-carrying wire.
  • Magnetic field due to currents (wire, loop, solenoid).
F = qvB sinθ  ·  F = BIL sinθ
CH 6Electromagnetic Induction▶

How a changing magnetic field generates electricity — the basis of generators.

Key concepts
  • Magnetic flux.
  • Faraday's law of induction.
  • Lenz's law (direction of induced current).
  • Generators and transformers.
ε = −dΦ/dt
CH 7Electromagnetic Waves▶

Self-propagating waves of electric and magnetic fields — from radio to gamma rays.

Key concepts
  • Nature and properties of EM waves.
  • The electromagnetic spectrum.
  • Speed, wavelength and frequency relationship.
c = fλ ≈ 3×10⁸ m/s
CH 8Light — Reflection & Refraction▶

How light bounces off surfaces and bends when it changes medium.

Key concepts
  • Laws of reflection; mirrors.
  • Refraction and Snell's law.
  • Refractive index and total internal reflection.
n₁ sinθ₁ = n₂ sinθ₂
CH 9Lenses▶

How converging and diverging lenses form images.

Key concepts
  • Convex and concave lenses.
  • Ray diagrams and image formation.
  • The lens equation and magnification.
1/f = 1/v − 1/u  ·  m = v/u
CH 10Interference & Diffraction▶

Wave behaviour of light: how light waves combine and spread.

Key concepts
  • Constructive and destructive interference.
  • Young's double-slit experiment.
  • Diffraction and the diffraction grating.
d sinθ = mλ (bright fringes)
CH 11Quantum Theory of the Atom▶

The quantum picture: energy comes in discrete packets, and atoms have energy levels.

Key concepts
  • Photoelectric effect and photons.
  • Energy of a photon; wave–particle duality.
  • Bohr model and atomic energy levels/spectra.
E = hf  ·  photoelectric: hf = W₀ + ½mv²max
CH 12Nuclear Physics▶

Inside the nucleus: its structure, stability and radioactivity.

Key concepts
  • Nucleons, isotopes, mass number & atomic number.
  • Radioactivity: alpha, beta, gamma decay.
  • Half-life and radioactive decay.
  • Mass defect and binding energy.
N = N₀ (½)^{t/T½}  ·  E = mc²
CH 13Nuclear Reactions▶

How nuclei change and release energy.

Key concepts
  • Nuclear fission and chain reactions.
  • Nuclear fusion.
  • Energy released in nuclear reactions.
  • Applications: reactors and nuclear energy.

Fundamentals of Computing

Programming in C · Semester 3

No experience needed. You'll go from understanding computers to writing real programs in the C language — variables, input/output, decisions, loops, files and functions. The secret to passing: practise coding every week.

C languageProblem solvingHands-on coding
TOPIC 1Introduction to IT▶

The big picture of computing before you start coding.

Key concepts
  • What a computer is; hardware vs software.
  • Input, processing, output, storage.
  • Data representation (bits, bytes, binary basics).
  • Types of software and operating systems.
TOPIC 2Programming Concepts▶

How to think like a programmer before writing any code.

Key concepts
  • Algorithms — step-by-step problem solving.
  • Flowcharts and pseudocode.
  • The program development cycle.
  • Programming languages and compilers.
TOPIC 3Programming Environment▶

Setting up and using the tools to write and run C programs.

Key concepts
  • Structure of a C program (main function, headers).
  • Writing, compiling and running code.
  • Comments, syntax and fixing errors.
  • Your first printf program.
#include <stdio.h>
int main(){ printf("Hello, world!"); return 0; }
TOPIC 4Elementary Programming (Parts 1 & 2)▶

The building blocks: storing and working with data.

Key concepts
  • Variables and data types (int, float, char, …).
  • Constants and naming rules.
  • Arithmetic, assignment and operators.
  • Operator precedence and type conversion.
TOPIC 5Input & Output▶

Communicating with the user — reading input and displaying results.

Key concepts
  • printf() for formatted output.
  • scanf() for reading input.
  • Format specifiers (%d, %f, %c, %s).
  • Escape sequences (\n, \t).
TOPIC 6Branching & Loops (Parts 1 & 2)▶

Making decisions and repeating actions — where programs become powerful.

Key concepts
  • Relational and logical operators.
  • Branching: if, if–else, switch.
  • Loops: for, while, do–while.
  • break, continue and nested loops.
TOPIC 7Input & Output Files▶

Saving and loading data so it persists after the program ends.

Key concepts
  • File pointers and opening modes (read/write/append).
  • fopen(), fclose().
  • Reading/writing with fscanf() / fprintf().
  • Checking for end-of-file and errors.
TOPIC 8Functions (Parts 1 & 2)▶

Breaking a program into reusable, named blocks — the key to clean code.

Key concepts
  • Defining and calling functions.
  • Parameters, arguments and return values.
  • Function prototypes.
  • Local vs global variables; scope.
int add(int a, int b){ return a + b; }
