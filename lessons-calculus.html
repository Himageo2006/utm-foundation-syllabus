Limits & Continuity

A limit tells you what value a function gets close to as x gets close to some number — even if the function isn't actually defined there. Limits are the foundation of all calculus.

Lesson 1.1 — What is a limit?▶

Consider this function, which is not defined at x = 1 (you'd divide by zero):

f(x) = (x − 1) / (x² − 1)

Even though we can't plug in x = 1, we can ask: what value does f(x) get close to as x gets close to 1? Build a table approaching 1 from both sides:

From the left (x < 1)

x = 0.9 → 0.5263  ·  0.99 → 0.5025  ·  0.999 → 0.5003

From the right (x > 1)

x = 1.1 → 0.4762  ·  1.01 → 0.4975  ·  1.001 → 0.4998

Both sides head toward 0.5. So we write:

limx→1 (x−1)/(x²−1) = 0.5
The small − means approaching from the left; the small + means from the right. These are called one-sided limits.
Lesson 1.2 — When does a limit exist?▶

The key rule:

The limit exists only if the left-hand limit and the right-hand limit are equal.
If   limx→a⁺ f(x) = limx→a⁻ f(x) = L   ⇒   limx→a f(x) = L

If the two sides give different values, the limit does not exist (DNE).

WORKED EXAMPLE

Determine whether the limit exists for the piecewise function:

f(x) = { x − 1,  if x < 2 ;   x + 2,  if x ≥ 2 }

Find limx→2 f(x).

1Left side (use x − 1):   limx→2⁻ (x − 1) = 2 − 1 = 1
2Right side (use x + 2):   limx→2⁺ (x + 2) = 2 + 2 = 4
3Left (1) ≠ Right (4), so the limit does not exist.
Lesson 1.3 — Rules of limits (the fast way)▶

For "nice" functions you don't need a table — just substitute the value, using these rules:

  • limx→a c = c  (limit of a constant is itself)
  • limx→a x = a
  • limx→a xⁿ = aⁿ
  • Sum/difference, product, quotient and power of limits all split apart (as long as you don't divide by 0).
WORKED EXAMPLES
alimx→½ (2x + 7) = 2(½) + 7 = 8
blimx→2 5x²(x − 4) = 5(2)²(2 − 4) = 5·4·(−2) = −40
climx→π (x² − 4)/(x + 2) = factor top = (x−2)(x+2)/(x+2) = (x − 2) → π − 2
dlimx→−2 √(5x² − 4) = √(5(−2)² − 4) = √16 = 4
Lesson 1.4 — Tricky limits: factorize or rationalize (0/0)▶

Sometimes substituting gives 0/0 — an "indeterminate" form. Don't give up! Simplify first by factorizing or rationalizing (multiplying by the conjugate), then substitute.

Method A — Factorizing

EXAMPLE

Find limx→6 (x² − 36)/(x − 6).

1Substituting gives 0/0 → must factor.
2Factor the top: x² − 36 = (x − 6)(x + 6)
3Cancel (x − 6):  = limx→6 (x + 6)
4Substitute: 6 + 6 = 12

More factorizing results:  limx→0 (2x²+5x)/x = 5;   limx→−1 (x²+x)/(x²−1) = ½.

Method B — Rationalizing (when there's a square root)

EXAMPLE

Find limx→4 (x − 4)/(√x − 2).

1Substituting gives 0/0.
2Multiply top & bottom by the conjugate (√x + 2):
(x − 4)(√x + 2) / [(√x − 2)(√x + 2)] = (x − 4)(√x + 2) / (x − 4)
3Cancel (x − 4):  = (√x + 2)
4Substitute x = 4: √4 + 2 = 4

More rationalizing results:  limx→−1 (x+1)/(√(3−x) − 2) = −4;   limx→9 (3 − √x)/(x − 9) = −1/6.

Lesson 1.5 — Infinite limits (vertical asymptotes)▶

Sometimes, as x approaches a value, f(x) shoots up to +∞ or down to −∞. Then we say f has an infinite limit, and the line x = a is a vertical asymptote.

EXAMPLE

Evaluate limx→0 1/x.

1From the right (x→0⁺): 1/x → +∞
2From the left (x→0⁻): 1/x → −∞
3The two sides disagree, so limx→0 1/x does not exist.
EXAMPLE

Evaluate limx→0 2/x².

1x² is always positive, so both sides give 2/(tiny positive) → +∞.
2limx→0 2/x² = +∞ (still counts as "does not exist", but we note it grows without bound).
Rule for f(x)/g(x) where g→0: look at the sign of the top and which way the bottom approaches 0 to decide +∞ or −∞.
Lesson 1.6 — Limits at infinity (horizontal asymptotes)▶

Now let x get huge (x → ∞) or hugely negative (x → −∞). We're studying the end behaviour of the graph.

Key theorem: for any positive power r,  limx→∞ c/xʳ = 0.  (Big bottom ⇒ fraction shrinks to 0.)

The method for fractions

  1. Divide every term, top and bottom, by the highest power of x in the denominator.
  2. Apply the theorem (terms like 1/x, 1/x² → 0).
EXAMPLE

Find limx→∞ (2x − 1)/(x + 1).

1Highest power in bottom is x. Divide each term by x:
(2 − 1/x) / (1 + 1/x)
2As x→∞, the 1/x terms vanish: (2 − 0)/(1 + 0)
3= 2  → so y = 2 is a horizontal asymptote.
EXAMPLE

Find limx→∞ (1 + 2x³)/(3 − 2x³).

1Divide by x³: (1/x³ + 2)/(3/x³ − 2)
2→ (0 + 2)/(0 − 2) = −1
Shortcut for rational functions: compare degrees. Bottom bigger → limit 0. Equal degrees → ratio of leading coefficients. Top bigger → limit DNE (∞).
Lesson 1.7 — Continuity at a point▶

A function is continuous at x = a if you can draw it through that point without lifting your pen. Formally, all three conditions must hold:

  1. f(a) is defined (a is in the domain).
  2. limx→a f(x) exists (left = right).
  3. limx→a f(x) = f(a) (the limit equals the actual value).

If any condition fails, f is discontinuous at a.

WORKED EXAMPLE — finding k

Find k so that this function is continuous at x = 2:

f(x) = { (3x − 6)/(2x³ − 3x² − 4),  x ≠ 2 ;   k,  x = 2 }
1For continuity we need limx→2 f(x) = f(2) = k.
2Compute the limit (it's 0/0, so factor): 3x − 6 = 3(x − 2), and 2x³ − 3x² − 4 = (x − 2)(2x² + x + 2).
3Cancel (x − 2):  limx→2 3 / (2x² + x + 2)
4Substitute x = 2: 3 / (8 + 2 + 2) = 3/12 = ¼
5So k = ¼.
Lesson 1.8 — Types of discontinuity & continuity on an interval▶

Three types of discontinuity

  • Removable — there's a hole, but the limit exists (you could "fill" the point). e.g. (x²−x−2)/(x−2) at x = 2.
  • Jump — left and right limits both exist but differ; the graph jumps.
  • Infinite — the function shoots to ±∞ (a vertical asymptote).
Polynomials and rational functions are continuous everywhere in their domain. A rational function is only discontinuous where its denominator = 0.
EXAMPLE — interval continuity

State where f(x) = (x − 1)/(x² + 2x) is continuous.

1Set denominator = 0: x² + 2x = x(x + 2) = 0 → x = 0 and x = −2.
2It's continuous everywhere except those two points:
3(−∞, −2) ∪ (−2, 0) ∪ (0, ∞)
To check continuity on a closed interval [a, b], also check it's continuous from the right at a and from the left at b.

Differentiation

The derivative f′(x) measures the instantaneous rate of change — the slope of the tangent to the curve at each point.

Lesson 2.1 — The rules of differentiation▶

The rules you'll use most

  • Power rule: d/dx(xⁿ) = n·xⁿ⁻¹
  • Constant: d/dx(c) = 0
  • Product rule: (uv)′ = u′v + uv′
  • Quotient rule: (u/v)′ = (u′v − uv′)/v²
  • Chain rule: dy/dx = dy/du · du/dx
WORKED EXAMPLE — power rule

Differentiate f(x) = 3x⁴ − 5x² + 7.

1Differentiate term by term using n·xⁿ⁻¹.
23x⁴ → 12x³ ; −5x² → −10x ; 7 → 0
3f′(x) = 12x³ − 10x
WORKED EXAMPLE — chain rule

Differentiate y = (2x + 1)⁵.

1Let u = 2x+1, so y = u⁵. Then dy/du = 5u⁴ and du/dx = 2.
2dy/dx = 5u⁴ · 2 = 10(2x+1)⁴
3dy/dx = 10(2x + 1)⁴
Also learn: d/dx(sin x)=cos x, d/dx(cos x)=−sin x, d/dx(eˣ)=eˣ, d/dx(ln x)=1/x.

Applications of Differentiation

Use derivatives to find maximum/minimum points, analyse curves, and solve optimisation problems.

Lesson 3.1 — Maxima, minima & optimisation▶

The method

  1. Find f′(x) and set f′(x) = 0 → these are the critical points.
  2. Use the second derivative: f″(x) > 0 → minimum; f″(x) < 0 → maximum.
WORKED EXAMPLE

Find and classify the turning point of f(x) = x² − 4x + 1.

1f′(x) = 2x − 4. Set = 0 → 2x − 4 = 0 → x = 2.
2f(2) = 4 − 8 + 1 = −3, so the point is (2, −3).
3f″(x) = 2 > 0 → it's a minimum at (2, −3).
Optimisation problems (largest area, least cost) reduce to: write the quantity as a function, differentiate, set to zero, solve.

Integration

Integration reverses differentiation and adds up infinitely many small pieces — used for areas and totals.

Lesson 4.1 — Basic integration & substitution▶
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C  (n ≠ −1)  ·  ∫ₐᵇ f(x) dx = F(b) − F(a)
WORKED EXAMPLE — indefinite

Find ∫ (6x² + 4) dx.

1∫6x² dx = 6·x³/3 = 2x³ ; ∫4 dx = 4x
2= 2x³ + 4x + C (don't forget +C!)
WORKED EXAMPLE — definite

Evaluate ∫₁³ 2x dx.

1Antiderivative of 2x is x².
2= [x²]₁³ = 3² − 1² = 9 − 1 = 8

Substitution

When the integrand contains a function and its derivative, let u = inner function. e.g. ∫2x(x²+1)⁵ dx: let u = x²+1, du = 2x dx → ∫u⁵ du = u⁶/6 + C = (x²+1)⁶/6 + C.

Integration of Trigonometric Functions

Lesson 5.1 — Standard trig integrals▶
∫ sin x dx = −cos x + C  ·  ∫ cos x dx = sin x + C  ·  ∫ sec²x dx = tan x + C

For powers or products, use trig identities first (e.g. sin²x = ½(1 − cos 2x)).

WORKED EXAMPLE

Evaluate ∫₀^(π/2) cos x dx.

1Antiderivative of cos x is sin x.
2= [sin x]₀^(π/2) = sin(π/2) − sin(0) = 1 − 0 = 1
WORKED EXAMPLE — using an identity

Find ∫ sin²x dx.

1Replace sin²x = ½(1 − cos 2x).
2∫½(1 − cos 2x) dx = ½x − ¼ sin 2x + C
3= ½x − ¼ sin 2x + C
6

Applications of Integration

Lesson 6.1 — Area under a curve & between curves▶
Area under f = ∫ₐᵇ f(x) dx  ·  Area between = ∫ₐᵇ [top − bottom] dx
WORKED EXAMPLE

Find the area under y = x² between x = 0 and x = 3.

1Area = ∫₀³ x² dx = [x³/3]₀³
2= 3³/3 − 0 = 27/3 = 9 square units
Volumes of revolution: V = π∫ₐᵇ [f(x)]² dx (rotating about the x-axis).

Ordinary Differential Equations (ODEs)

Lesson 7.1 — Separable equations▶

An ODE links a function to its derivatives. The simplest to solve are separable: get all the y's on one side and x's on the other, then integrate both sides.

WORKED EXAMPLE

Solve dy/dx = 2x·y.

1Separate: (1/y) dy = 2x dx
2Integrate both sides: ln|y| = x² + C
3Solve for y: y = Ae^(x²)  (A = e^C). y = A·e^(x²)
A general solution has a constant (A or C); use an initial condition like y(0)=2 to find it.

Numerical Methods

Lesson 8.1 — Newton–Raphson & the trapezoidal rule▶

When exact answers are hard, approximate them.

Newton–Raphson (finding roots)

xₙ₊₁ = xₙ − f(xₙ)/f′(xₙ)
WORKED EXAMPLE

Use one Newton–Raphson step on f(x) = x² − 2, starting at x₀ = 1.5 (finding √2).

1f(x) = x² − 2, f′(x) = 2x. At x₀ = 1.5: f = 0.25, f′ = 3.
2x₁ = 1.5 − 0.25/3 = 1.5 − 0.0833 = 1.4167 (√2 ≈ 1.4142 — very close already).

Trapezoidal rule (approximate area)

∫ₐᵇ f(x) dx ≈ (h/2)[y₀ + 2y₁ + 2y₂ + … + yₙ],   h = (b−a)/n
Simpson's rule is similar but more accurate, using the pattern [1, 4, 2, 4, …, 4, 1].
