Introduction to IT

Computers, hardware & software▶

A computer follows the input → process → output model, with storage. Key ideas:

  • Hardware = physical parts (CPU, RAM, disk, I/O devices).
  • Software = programs (system software like the OS, and application software).
  • Data is stored in binary (bits → bytes).
Foundation: 8 bits = 1 byte; a bit is 0 or 1. Everything (text, images) is ultimately binary.

Programming Concepts

Algorithms, flowcharts & pseudocode▶

Before coding, plan the logic. An algorithm is a step-by-step recipe. We express it as a flowchart (boxes & arrows) or pseudocode (plain-English steps).

EXAMPLE — algorithm for "add two numbers"
1START
2READ a, b
3sum = a + b
4PRINT sum
5END

Programming Environment

Your first C program▶

A C program needs a main() function and headers. You compile it, then run it.

#include <stdio.h>

int main() {
    printf("Hello, world!\n");
    return 0;
}
  • #include <stdio.h> — brings in input/output functions.
  • printf — prints text. \n = new line.
  • return 0; — tells the OS the program ended successfully.

Elementary Programming

Variables, data types & operators▶

A variable stores a value. Each has a type:

  • int — whole numbers
  • float / double — decimals
  • char — a single character
int age = 19;
float price = 4.50;
char grade = 'A';
int total = age + 1;   // arithmetic
Operators: + − * / and % (remainder). Watch out: int/int drops the decimal (7/2 = 3, not 3.5).

Input & Output

printf & scanf▶

printf displays; scanf reads input. Format specifiers: %d int, %f float, %c char, %s string.

int n;
printf("Enter a number: ");
scanf("%d", &n);              // & gives scanf the variable's address
printf("You typed %d\n", n);
Common beginner bug: forgetting the & in scanf("%d", &n).

Branching & Loops

Decisions (if/switch) and repetition (for/while)▶

Branching — make a decision

if (mark >= 50) {
    printf("Pass\n");
} else {
    printf("Fail\n");
}

Loops — repeat

// print 1 to 5
for (int i = 1; i <= 5; i++) {
    printf("%d ", i);
}

Also: while (repeat while a condition is true) and do–while (runs at least once). break exits a loop, continue skips to the next round.

7

Files (Input & Output)

Reading & writing files▶

Files let data survive after the program ends. Open with fopen (mode: "r" read, "w" write, "a" append), then close with fclose.

FILE *fp = fopen("data.txt", "w");
fprintf(fp, "Score: %d\n", 95);
fclose(fp);

Read back with fscanf(fp, "%d", &x). Always check the file opened (if (fp == NULL) ...).

Functions

Reusable blocks of code▶

A function packages code you can call repeatedly. It can take parameters and return a value.

int add(int a, int b) {   // returns an int
    return a + b;
}

int main() {
    int sum = add(3, 4);  // sum = 7
    printf("%d\n", sum);
    return 0;
}
  • Define the function (or a prototype) before you use it.
  • Local variables live inside one function; global ones are shared.
WHY IT MATTERS

Functions keep programs short, readable and reusable — write the logic once, call it many times.
