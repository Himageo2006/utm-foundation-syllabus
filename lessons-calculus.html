Limits & Continuity

A limit tells you what value a function gets close to as x gets close to some number — even if the function isn't actually defined there. Limits are the foundation of all calculus.

Lesson 1.1 — What is a limit?▶

The limit of a function is the value f(x) heads toward as x gets closer and closer to some number — written

limx→a f(x) = L

and read "the limit of f(x), as x approaches a, equals L." The crucial idea: we care about the journey, not the destination. The function need not even be defined at a — we only watch what happens near a.

Why we need limits

Consider this function, which is not defined at x = 1 (substituting 1 gives 0/0, division by zero):

f(x) = (x − 1) / (x² − 1)

We can't plug in x = 1, but we can still ask: what value does f(x) get close to as x gets close to 1? Build a table approaching 1 from both sides:

From the left (x < 1, sliding up toward 1)

x = 0.9 → 0.5263  ·  0.99 → 0.5025  ·  0.999 → 0.5003

From the right (x > 1, sliding down toward 1)

x = 1.1 → 0.4762  ·  1.01 → 0.4975  ·  1.001 → 0.4998

Both columns squeeze toward the same number, 0.5. So even though f(1) doesn't exist, the limit does:

limx→1 (x−1)/(x²−1) = 0.5
The small − means approaching from the left (limx→a⁻); the small + means from the right (limx→a⁺). These are called one-sided limits.
WORKED EXAMPLE — the limit can differ from the value

Let g(x) = x + 3 for x ≠ 2, but g(2) = 10 (a single "misplaced" point). Find limx→2 g(x).

1Near x = 2 the rule is g(x) = x + 3, so the values head to 2 + 3 = 5.
2The actual point g(2) = 10 is irrelevant — the limit ignores it.
3limx→2 g(x) = 5, even though g(2) = 10. Limit ≠ value here.
Three things to keep straight: the left limit, the right limit, and the actual value f(a). They can all be different — limits only look at the first two.
Lesson 1.2 — When does a limit exist?▶

The key rule:

The limit exists only if the left-hand limit and the right-hand limit are equal.
If   limx→a⁺ f(x) = limx→a⁻ f(x) = L   ⇒   limx→a f(x) = L

If the two sides give different values, the two-sided limit does not exist (DNE) — even though each one-sided limit is perfectly fine on its own.

WORKED EXAMPLE 1 — limit does NOT exist (a jump)

Determine whether the limit exists for the piecewise function:

f(x) = { x − 1,  if x < 2 ;   x + 2,  if x ≥ 2 }

Find limx→2 f(x).

1Left side (use x − 1):   limx→2⁻ (x − 1) = 2 − 1 = 1
2Right side (use x + 2):   limx→2⁺ (x + 2) = 2 + 2 = 4
3Left (1) ≠ Right (4), so the limit does not exist.
WORKED EXAMPLE 2 — limit DOES exist

Same idea, but a function that matches up at the join:

h(x) = { x² ,  if x ≤ 3 ;   6x − 9,  if x > 3 }

Find limx→3 h(x).

1Left side (use x²):   limx→3⁻ x² = 3² = 9
2Right side (use 6x − 9):   limx→3⁺ (6x − 9) = 18 − 9 = 9
3Both sides agree, so limx→3 h(x) = 9 (the limit exists).
WORKED EXAMPLE 3 — reading a graph

A common exam style: you're given a graph and asked to read limits at a point where there's an open circle at (4, 2) and a filled dot at (4, 5).

1Both branches of the curve approach height 2 as x → 4 from each side ⇒ limx→4 f(x) = 2.
2The filled dot says f(4) = 5 — that's the value, not the limit.
3So limx→4 f(x) = 2 while f(4) = 5. (The graph is discontinuous here — see Lesson 1.7.)
Lesson 1.3 — Rules of limits (the fast way)▶

For "nice" (continuous) functions you don't need a table — just substitute the value. These limit laws make it legal:

  • Constant: limx→a c = c
  • Identity: limx→a x = a,   and limx→a xⁿ = aⁿ
  • Constant multiple: limx→a [c·f(x)] = c·lim f(x)
  • Sum / difference: lim [f ± g] = lim f ± lim g
  • Product: lim [f · g] = (lim f)(lim g)
  • Quotient: lim [f / g] = (lim f)/(lim g), provided lim g ≠ 0
  • Power / root: lim [f(x)]ⁿ = (lim f)ⁿ, and the same for nth roots

In short: for polynomials, and for rational/root functions where the denominator isn't 0, direct substitution just works.

WORKED EXAMPLES
alimx→½ (2x + 7) = 2(½) + 7 = 8
blimx→2 5x²(x − 4) = 5(2)²(2 − 4) = 5·4·(−2) = −40
climx→3 (x² + 2x − 1) = 9 + 6 − 1 = 14  (plain polynomial → substitute)
dlimx→1 (2x + 3)/(x + 4) = (2 + 3)/(1 + 4) = 5/5 = 1  (bottom ≠ 0, so substitute)
elimx→π (x² − 4)/(x + 2) = factor top = (x−2)(x+2)/(x+2) = (x − 2) → π − 2
flimx→−2 √(5x² − 4) = √(5(−2)² − 4) = √16 = 4
⚠️ Substitution only fails when you hit 0/0 (an indeterminate form). Then switch to factorizing or rationalizing — that's exactly what Lesson 1.4 is about.
Lesson 1.4 — Tricky limits: factorize or rationalize (0/0)▶

Sometimes substituting gives 0/0 — an "indeterminate" form. Don't give up! Simplify first by factorizing or rationalizing (multiplying by the conjugate), then substitute.

Method A — Factorizing

EXAMPLE

Find limx→6 (x² − 36)/(x − 6).

1Substituting gives 0/0 → must factor.
2Factor the top: x² − 36 = (x − 6)(x + 6)
3Cancel (x − 6):  = limx→6 (x + 6)
4Substitute: 6 + 6 = 12

More factorizing results:  limx→0 (2x²+5x)/x = 5;   limx→−1 (x²+x)/(x²−1) = ½.

Method B — Rationalizing (when there's a square root)

EXAMPLE

Find limx→4 (x − 4)/(√x − 2).

1Substituting gives 0/0.
2Multiply top & bottom by the conjugate (√x + 2):
(x − 4)(√x + 2) / [(√x − 2)(√x + 2)] = (x − 4)(√x + 2) / (x − 4)
3Cancel (x − 4):  = (√x + 2)
4Substitute x = 4: √4 + 2 = 4

More rationalizing results:  limx→−1 (x+1)/(√(3−x) − 2) = −4;   limx→9 (3 − √x)/(x − 9) = −1/6.

Lesson 1.5 — Infinite limits (vertical asymptotes)▶

Sometimes, as x approaches a value, f(x) shoots up to +∞ or down to −∞. Then we say f has an infinite limit, and the line x = a is a vertical asymptote.

EXAMPLE

Evaluate limx→0 1/x.

1From the right (x→0⁺): 1/x → +∞
2From the left (x→0⁻): 1/x → −∞
3The two sides disagree, so limx→0 1/x does not exist.
EXAMPLE

Evaluate limx→0 2/x².

1x² is always positive, so both sides give 2/(tiny positive) → +∞.
2limx→0 2/x² = +∞ (still counts as "does not exist", but we note it grows without bound).
Rule for f(x)/g(x) where g→0: look at the sign of the top and which way the bottom approaches 0 to decide +∞ or −∞.
Lesson 1.6 — Limits at infinity (horizontal asymptotes)▶

Now let x get huge (x → ∞) or hugely negative (x → −∞). We're studying the end behaviour of the graph.

Key theorem: for any positive power r,  limx→∞ c/xʳ = 0.  (Big bottom ⇒ fraction shrinks to 0.)

The method for fractions

  1. Divide every term, top and bottom, by the highest power of x in the denominator.
  2. Apply the theorem (terms like 1/x, 1/x² → 0).
EXAMPLE

Find limx→∞ (2x − 1)/(x + 1).

1Highest power in bottom is x. Divide each term by x:
(2 − 1/x) / (1 + 1/x)
2As x→∞, the 1/x terms vanish: (2 − 0)/(1 + 0)
3= 2  → so y = 2 is a horizontal asymptote.
EXAMPLE

Find limx→∞ (1 + 2x³)/(3 − 2x³).

1Divide by x³: (1/x³ + 2)/(3/x³ − 2)
2→ (0 + 2)/(0 − 2) = −1
Shortcut for rational functions: compare degrees. Bottom bigger → limit 0. Equal degrees → ratio of leading coefficients. Top bigger → limit DNE (∞).
Lesson 1.7 — Continuity at a point▶

A function is continuous at x = a if you can draw it through that point without lifting your pen. Formally, all three conditions must hold:

  1. f(a) is defined (a is in the domain).
  2. limx→a f(x) exists (left = right).
  3. limx→a f(x) = f(a) (the limit equals the actual value).

If any condition fails, f is discontinuous at a.

WORKED EXAMPLE — finding k

Find k so that this function is continuous at x = 2:

f(x) = { (3x − 6)/(2x³ − 3x² − 4),  x ≠ 2 ;   k,  x = 2 }
1For continuity we need limx→2 f(x) = f(2) = k.
2Compute the limit (it's 0/0, so factor): 3x − 6 = 3(x − 2), and 2x³ − 3x² − 4 = (x − 2)(2x² + x + 2).
3Cancel (x − 2):  limx→2 3 / (2x² + x + 2)
4Substitute x = 2: 3 / (8 + 2 + 2) = 3/12 = ¼
5So k = ¼.
Lesson 1.8 — Types of discontinuity & continuity on an interval▶

Three types of discontinuity

  • Removable — there's a hole, but the limit exists (you could "fill" the point). e.g. (x²−x−2)/(x−2) at x = 2.
  • Jump — left and right limits both exist but differ; the graph jumps.
  • Infinite — the function shoots to ±∞ (a vertical asymptote).
Polynomials and rational functions are continuous everywhere in their domain. A rational function is only discontinuous where its denominator = 0.
EXAMPLE — interval continuity

State where f(x) = (x − 1)/(x² + 2x) is continuous.

1Set denominator = 0: x² + 2x = x(x + 2) = 0 → x = 0 and x = −2.
2It's continuous everywhere except those two points:
3(−∞, −2) ∪ (−2, 0) ∪ (0, ∞)
To check continuity on a closed interval [a, b], also check it's continuous from the right at a and from the left at b.

Differentiation

The derivative f′(x) measures the instantaneous rate of change — the slope of the tangent to the curve at each point.

Lesson 2.1 — Differentiation from First Principle▶

The first principle (the definition of differentiation):

f′(x) = limδx→0 [f(x + δx) − f(x)] / δx,   when the limit exists.

The 4-step procedure

  1. Given y = f(x), write the expression f(x + δx).
  2. Form the difference f(x + δx) − f(x).
  3. Divide by δx and simplify: [f(x + δx) − f(x)] / δx.
  4. Take the limit as δx → 0.
WORKED EXAMPLE — y = 2x²
1f(x + δx) = 2(x + δx)² = 2x² + 4x·δx + 2(δx)²
2f(x + δx) − f(x) = 4x·δx + 2(δx)²
3÷ δx → 4x + 2δx
4limδx→0 (4x + 2δx) = 4x
Other results from first principle: y = 1/x → f′(x) = −1/x²;   y = √(x+5) → f′(x) = 1/(2√(x+5)) (multiply by the conjugate);   y = √(2x−1) → 1/√(2x−1).
Lesson 2.2 — Differentiating Simple Algebraic Functions▶
  • Constant: if y = c then dy/dx = 0.
  • Power rule: if y = xⁿ then dy/dx = n·xⁿ⁻¹.
  • Basic table: d/dx(mx+c)=m, d/dx(xᵃ)=axᵃ⁻¹, d/dx(eˣ)=eˣ, d/dx(ln x)=1/x.
  • Rewrite roots/fractions as powers first: 1/xⁿ = x⁻ⁿ, √x = x^(1/2), ∛x = x^(1/3).
WORKED EXAMPLES
1y = x¹⁵ → 15x¹⁴
2y = 4x³ + 2x² + 3x → 12x² + 4x + 3
3y = 1/x⁵ − 2/x³ + x³/2 → −5/x⁶ + 6/x⁴ + 3x²/2
4y = 3√x + ∛x → 3/(2√x) + 1/(3∛(x²))
Differentiation of multiples & sums: (cu)′ = c·u′; (u + v)′ = u′ + v′. e.g. y = 2tan x − 3cos x → 2sec²x + 3sin x.
Lesson 2.3 — The Chain Rule▶

For a composite function y = f(g(x)), let u = g(x). Then:

dy/dx = dy/du · du/dx
WORKED EXAMPLES
ay = (x³+6x−3)²⁰: u = x³+6x−3 → 20(x³+6x−3)¹⁹(3x²+6)
by = sin(2x−5) → 2cos(2x−5)
cy = e^(3x+1) → 3e^(3x+1)
dy = ln 3x → (1/3x)·3 = 1/x
Lesson 2.4 — Product & Quotient Rules▶

Product rule

if y = uv,   dy/dx = u·v′ + v·u′
EXAMPLES
ay = x³cos x → −x³sin x + 3x²cos x
by = 5x sin x → 5x cos x + 5 sin x
cy = x²ln x → x + 2x ln x

Quotient rule

if y = u/v (v ≠ 0),   dy/dx = (v·u′ − u·v′)/v²
EXAMPLES
ay = 1/(1−sin x) → cos x/(1−sin x)²
by = (2x+5)/(3x+2) → −11/(3x+2)²
Function-value example: given f(1)=−2, f′(1)=3, g(1)=5, g′(1)=−4 — for y=(x−2)f(x), dy/dx at x=1 = (1−2)(3)+(−2)(1) = −5; for y=f(x)/g(x), = [5(3)−(−2)(−4)]/5² = 7/25.
Lesson 2.5 — Higher-Order Derivatives▶

Differentiate repeatedly: y′ (first), y″ = d²y/dx² (second), y‴, y⁽⁴⁾, … y⁽ⁿ⁾.

EXAMPLE

f(x) = 4x⁴ − 2x³ + 3 — first five derivatives:

1f′ = 16x³ − 6x²; f″ = 48x² − 12x; f‴ = 96x − 12
2f⁽⁴⁾ = 96; f⁽⁵⁾ = 0, so f⁽ⁿ⁾ = 0 for n ≥ 5.
For y = x²/(x−2): dy/dx = (x²−4x)/(x−2)², and d²y/dx² = 8/(x−2)³.
Lesson 2.6 — Exponential & Logarithmic Functions▶

Exponential

  • if y = eᵘ, then dy/dx = eᵘ·(du/dx). e.g. y = e^(x²+1) → 2x·e^(x²+1).
  • if y = aᵘ, then dy/dx = aᵘ·ln a·(du/dx). e.g. y = 10^(2x−5) → 2·10^(2x−5)·ln 10.

Logarithmic

  • if y = ln u, then dy/dx = (1/u)·(du/dx). e.g. y = 3ln(x²+2x) → 6(x+1)/[x(x+2)].
  • if y = logₐ u, then dy/dx = (1/u)·logₐe·(du/dx) = (1/(u ln a))·(du/dx). e.g. y = log₂x⁵ → 5/(x ln 2).
  • Use log laws to split first: ln[(3x+1)/(x+1)] = ln(3x+1) − ln(x+1); ln√(...) = ½ln(...).
d/dx(ln eˣ) = d/dx(x) = 1, since ln eˣ = x ln e = x.
Lesson 2.7 — Trigonometric Functions▶
STANDARD DERIVATIVES (chain-rule form, u = f(x))
sind/dx sin u = cos u · u′
cosd/dx cos u = −sin u · u′
tand/dx tan u = sec²u · u′
cotd/dx cot u = −cosec²u · u′
secd/dx sec u = sec u tan u · u′
cscd/dx cosec u = −cosec u cot u · u′
EXAMPLES
ay = 5x + sin x → 5 + cos x
by = cos(x²−3) → −2x sin(x²−3)
cy = cos²(x²−3) → −2x sin 2(x²−3) (using sin 2a = 2 sin a cos a)
dy = x²tan 2x → 2x²sec²2x + 2x tan 2x (product rule)
Lesson 2.8 — Implicit Differentiation▶

Some relations (e.g. x² + y² = 25 or x³ + y³ = 6xy) can't easily be solved for y. Instead, differentiate both sides with respect to x — and whenever you differentiate a term in y, multiply by dy/dx (chain rule) — then solve for dy/dx.

EXAMPLE — x² + y² = 25
1d/dx: 2x + 2y(dy/dx) = 0
2Solve: dy/dx = −x/y
EXAMPLE — 3y² − 2x² = 2xy (product rule on the right)
16y(dy/dx) − 4x = 2x(dy/dx) + 2y
2(6y − 2x)(dy/dx) = 2y + 4x → dy/dx = (y + 2x)/(3y − x)
Logarithmic differentiation handles y = xˣ: take ln of both sides → ln y = x ln x; differentiate → (1/y)y′ = 1 + ln x; so dy/dx = xˣ(1 + ln x).
Lesson 2.9 — Parametric Differentiation▶

If x = h(t) and y = g(t), then:

dy/dx = (dy/dt) · (dt/dx) = (dy/dt) / (dx/dt)
EXAMPLE — x = 2t, y = 4 − 4t − 4t²
1dy/dt = −4 − 8t, dx/dt = 2
2dy/dx = (−4 − 8t)·(1/2) = −2 − 4t

For the second derivative, let w = dy/dx and use d²y/dx² = (dw/dt)·(dt/dx). For x = (t−3)/t, y = (t²+4)/t at t = 1: dy/dx = ⅓(t²−4) = −1, and d²y/dx² = (2/9)t³ = 2/9.

Applications of Differentiation

Use derivatives to find rates of change, tangents and normals, maximum/minimum points, sketch curves, and solve optimisation problems.

Lesson 3.1 — Rate of Change▶

If y is a function of x, then dy/dx is the rate of change of y with respect to x (always state appropriate units). For time problems, chain related rates together:

dr/dt = (dr/du) · (du/dt)   and   dr/dt = (change in r) / (change in t)
WORKED EXAMPLE — water in a container

V = 1.5h² + 8h, dV/dt = 12 cm³/s. Find dh/dt when h = 8.

1dV/dh = 3h + 12. So dh/dt = (dh/dV)·(dV/dt) = [1/(3h+12)]·12.
2At h = 8: 12/(24+12) = 12/36 = ⅓ cm/s
Sign matters: "increasing" → positive rate, "decreasing" → negative. Example: sphere with dr/dt = 2, find dV/dt when A = 8 → dV/dt = 4πr²·2 = 8πr²; with 4πr² = 8 → r² = 2/π → dV/dt = 16 ml³/s.
Lesson 3.2 — Tangents & Normals to a Curve▶
  • The gradient of the tangent at a point is dy/dx evaluated there. Equation of a line: y − y₀ = m(x − x₀).
  • The normal is perpendicular: (gradient of tangent) × (gradient of normal) = −1, so mnormal = −1/mtangent.
TANGENT — y = x² − 5x + 3 at x = 3
1y(3) = 9 − 15 + 3 = −3, so point (3, −3).
2dy/dx = 2x − 5; at x = 3 → m = 1.
3y − (−3) = 1(x − 3) → y = x − 6
NORMAL — y = 2x³ − 3x² + 5 at (2, 9)
1dy/dx = 6x² − 6x; at x = 2 → mtan = 12, so mnormal = −1/12.
2y − 9 = −(1/12)(x − 2) → 12y + x = 110
For implicit curves (e.g. the ellipse 3x² + 4y² = 48 at (2,3)), differentiate implicitly to get dy/dx = −6x/8y = −½, then write the tangent x + 2y = 8.
Lesson 3.3 — Increasing, Decreasing & Critical Points▶
  • f′(x) > 0 → function increasing; f′(x) < 0 → decreasing; f′(x) = 0 → stationary.
  • A critical point (x₀, f(x₀)) occurs where f′(x₀) = 0 or f′(x₀) is undefined (but x₀ must be in the domain of f).
EXAMPLE — f(x) = 4x − x²
1f′(x) = 4 − 2x. Increasing where 4 − 2x > 0 → x < 2.
2Decreasing where 4 − 2x < 0 → x > 2.
Use a sign table: for f(x) = 2x³ − 3x² − 12x + 1, f′(x) = 6x² − 6x − 12 = 6(x−2)(x+1) — increasing for x < −1 or x > 2, decreasing for −1 < x < 2.
Lesson 3.4 — First & Second Derivative Tests▶

First Derivative Test — examine the sign of f′ around the critical number:

  • + → 0 → −  means a relative maximum (convex, ∩).
  • − → 0 → +  means a relative minimum (concave, ∪).

Second Derivative Test:

  • f″(x₀) < 0 → convex → local maximum.
  • f″(x₀) > 0 → concave → local minimum.
  • f″(x₀) = 0 or undefined → test fails; fall back to the First Derivative Test.
EXAMPLE — y = 4 + 12x − 3x² − 2x³
1f′(x) = 12 − 6x − 6x² = 0 → x = 1 or x = −2.
2f″(x) = −6 − 12x. At x = 1: −18 < 0 → max (1, 11). At x = −2: 18 > 0 → min (−2, −16).
A point of inflection is where the curve changes concavity, found where f″(x) = 0. For the curve above, f″(x) = 0 → x = −½, giving inflection point (−½, −5/2).
Lesson 3.5 — Curve Sketching & Asymptotes▶

The 7-step procedure:

  1. Find the x- and y-intercepts.
  2. Find f′(x) and f″(x).
  3. Critical numbers: solve f′(x) = 0.
  4. Second derivative test for local max/min.
  5. Inflection points: solve f″(x) = 0 and test concavity.
  6. Find horizontal & vertical asymptotes.
  7. Sketch using all the information.
Vertical asymptote: set the denominator of a rational function = 0. Horizontal asymptote: take limx→±∞ f(x) by dividing by the highest power of x in the denominator. e.g. f(x) = (4−5x)/(1−x) has vertical asymptote x = 1 and horizontal asymptote y = 5.
Lesson 3.6 — Absolute Max/Min & Optimisation▶

Absolute extrema on a closed interval [a, b]

Find critical numbers inside the interval, then compare f at those points and at the endpoints a and b — the largest is the absolute maximum, the smallest the absolute minimum.

EXAMPLE — f(x) = 2x³ − x² + 2 on −2 ≤ x ≤ 1
1f′(x) = 6x² − 2x = 0 → x = 0, ⅓.
2Values: f(−2)=−18, f(0)=2, f(⅓)=53/27, f(1)=3.
3Absolute max = 3 (at x=1); absolute min = −18 (at x=−2) — both at endpoints.

Optimisation

Write the quantity as a function of one variable (use a constraint to eliminate the other), differentiate, set f′ = 0, then use the second derivative test (f″ > 0 min, f″ < 0 max).

Example: closed cylinder holding 600 cm³ → A = 2πr² + 1200/r. A′ = 4πr − 1200/r² = 0 → r ≈ 4.57 cm, h ≈ 9.14 cm, minimum area ≈ 393.81 cm². Open square-base box, surface 144 cm² → V = 36x − ¼x³, V′ = 0 → x = √48, max volume ≈ 166.08 cm³.

Integration

Integration is the reverse process of differentiation. If dy/dx = f(x), then ∫ f(x) dx = y. Integration is more challenging than differentiation — it is not always obvious which technique to use.

Lesson 4.1 — Introduction & notation▶
∫ 4x dx = 2x² + C,   C ∈ ℝ

Each part of the notation has a name:

∫integral sign
4xthe integrand (function being integrated)
dxalways written — shows the variable of integration
Cconstant of integration

Integrals of this type are indefinite integrals (as opposed to definite integrals).

Why the "+C"? If y = x², y = x²+3, y = x²−44 all differentiate to dy/dx = 2x. The constant disappears on differentiating, so on integrating we don't know which it was — write +C. An indefinite integral must always include +C.
Differentiating decreases the order of the polynomial by one; integrating increases it by one.
Lesson 4.2 — Table of integration formulas▶
1. ∫ xⁿ dx = xⁿ⁺¹/(n+1) + C  (n ≠ −1)
2. ∫ (1/x) dx = ln|x| + C
3. ∫ eˣ dx = eˣ + C
4. ∫ aˣ dx = aˣ/ln a + C
∫ sin x dx = −cos x  ·  ∫ cos x dx = sin x
∫ sec²x dx = tan x  ·  ∫ csc²x dx = −cot x
∫ sec x tan x dx = sec x  ·  ∫ csc x cot x dx = −csc x
∫ sec x dx = ln|sec x + tan x|  ·  ∫ csc x dx = ln|csc x − cot x|
∫ tan x dx = ln|sec x|  ·  ∫ cot x dx = ln|sin x|

Strategy for integration: 1) Simplify the integrand. 2) Look for an obvious substitution. 3) Classify by form. 4) Try again.

Lesson 4.6 — Definite integrals & properties▶
∫ₐᵇ f(x) dx = [F(x)]ₐᵇ = F(b) − F(a)  where F′(x) = f(x)

Integration of Trigonometric Functions

Three families: (1) odd powers of sin/cos, (2) even powers of sin/cos, (3) products sin ax cos bx etc. Plus integration of irrational functions by trig substitution. Key identity: sin²x + cos²x = 1; note cos(−x) = cos x, but sin(−x) ≠ sin x.

Lesson 5.1 — Odd powers of sin x / cos x▶

For ∫ sinᵐx cosⁿx dx where m is odd: split off one sin x, convert the rest using sin²x = 1 − cos²x, then let u = cos x. If n is odd: split off one cos x, use cos²x = 1 − sin²x, let u = sin x.

Two main applications: (1) area of a region — under a curve or between two curves; (2) volume of a solid of revolution.

Lesson 6.1 — Area under a curve (x-axis)▶
Above x-axis: A = ∫ₐᵇ f(x) dx  ·  Below x-axis: A = |∫ₐᵇ g(x) dx| (use modulus)
WORKED EXAMPLE

Area bounded by y = 2 − x², the x-axis, x = 0 and x = 1.

1∫₀¹ (2 − x²) dx = [2x − x³/3]₀¹ = 2 − 1/3

1∫₀¹ x³ dx = 1/4 (above); |∫₋₁⁰ x³ dx| = 1/4 (below)
2Total = 1/4 + 1/4 = 1/2 square units

Ordinary Differential Equations (ODEs)

Lesson 7.1 — Introduction & basic concepts▶

An Ordinary Differential Equation (ODE) is an equation involving derivatives of a function y with respect to a single independent variable x. They lie at the core of physics — Newton's laws, Maxwell's equations, Schrödinger's equation are all differential equations.

REAL-WORLD ODEs (from the slides)
1Radioactive decay: du/dt = −k·u(t), k > 0 — amount of material decreases over time.
2Newton's law of cooling: dθ/dt = −k(θ − θ₀) — a hot object cools toward its surroundings θ₀.

Order of an ODE

The order is the highest derivative that appears.

a) dy/dx − 2y = eˣ → order 1  ·  b) d²y/dx² − 4 dy/dx + 3y → order 2  ·  c) eˣ dy/dx + xy² = 0 → order 1
An equation that contains at least one of dy/dx, d²y/dx², …, dⁿy/dxⁿ is an ODE. Here x is the independent variable and y is a function of x.
Lesson 7.2 — Formation of differential equations▶

A differential equation can be formed by eliminating the arbitrary constants from an algebraic equation. Differentiate as many times as there are constants, then eliminate them.

WORKED EXAMPLE — one constant

Form an ODE from y = x + A/x.

1Multiply by x: xy = x² + A.
2Differentiate implicitly: x dy/dx + y = 2x. x dy/dx + y = 2x
WORKED EXAMPLE — eliminating A

Form an ODE from y = sin x + A/sin x.

1Multiply by sin x: y sin x = sin²x + A.
2Differentiate: dy/dx·sin x + y cos x = 2 sin x cos x.
3Rearrange: tan x dy/dx + y = 2 sin x (first order).
WORKED EXAMPLE — two constants → 2nd order

Form an ODE from y = e^(−2t)(A cos 3t + B sin 3t).

1Differentiate twice; A and B cancel using the original relation.
2Result: d²y/dt² + 4 dy/dt + 13y = 0 (second order — two constants give order 2).
Number of arbitrary constants = order of the resulting ODE.
Lesson 7.3 — Solutions of ODEs (general, particular, IVP)▶

y = φ(x) is a solution if substituting it produces an identity for all x in an interval.

  • A solution containing arbitrary constants is a general solution.
  • Fixing the constants gives a particular solution.
  • An ODE together with initial conditions is an Initial Value Problem (IVP).
WORKED EXAMPLE — verify a solution

Show xy = 2(x − 1) solves (x² − x) dy/dx = y.

1Differentiate: x dy/dx + y = 2 ⟹ dy/dx = (2 − y)/x.
2Substitute: (x² − x)(2 − y)/x = (x − 1)(2 − y) = 2x − xy − 2 + y = 2x − 2(x − 1) − 2 + y = y. ✓
WORKED EXAMPLE — find the constant

y = 1 + A·e^(−sin x) solves dy/dx = (1 − y)cos x. Find A if y = 2 when x = 0.

1At x = 0: 2 = 1 + A·e⁰ ⟹ A = 1, so y = 1 + e^(−sin x).
Lesson 7.4 — Separable variables equations▶

dy/dx = f(x,y) is separable when f(x,y) = u(x)·v(y), so it rearranges to dy/v(y) = u(x) dx. Then integrate both sides; apply the initial condition if given.

dy/dx + xy = x ⟹ dy/(1−y) = x dx ✓ separable. But dy/dx = (x+y)/x = 1 + y/x ✗ not separable.
WORKED EXAMPLE

Solve dy/dx = 3x/(y − 2).

1Separate: (y − 2) dy = 3x dx.
2Integrate: y²/2 − 2y = (3/2)x² + c ⟹ y² − 4y = 3x² + k.
3Complete the square: (y − 2)² = 3x² + C ⟹ y = 2 + √(3x² + C).
WORKED EXAMPLE

Solve cot y dy/dx = cot x.

1(cos y/sin y) dy = (cos x/sin x) dx ⟹ ln|sin y| = ln|sin x| + k.
2sin y = C sin x (C = eᵏ).
WORKED EXAMPLE — IVP with partial fractions

Solve dy/dx = y/(x² − 1), y(2) = 1.

1(1/y) dy = dx/(x² − 1); split 1/(x²−1) = ½/(x−1) − ½/(x+1).
2ln y = ½ln|x−1| − ½ln|x+1| + C; at (2,1): C = ½ln 3.
3y = √[3(x−1)/(x+1)].
Lesson 7.5 — Linear equations & the integrating factor▶

A first-order linear ODE: a(x) dy/dx + b(x) y = c(x). Procedure:

1) Standard form dy/dx + P(x)y = Q(x)   2) ∫P dx   3) Integrating factor ρ = e^(∫P dx)   4) d/dx(ρy) = ρQ   5) Integrate
WORKED EXAMPLE

Solve dy/dx + y/x = x³.

1P = 1/x, ∫P dx = ln x, ρ = e^(ln x) = x.
2d/dx(xy) = x³·x = x⁴ ⟹ xy = x⁵/5 + c.
3y = x⁴/5 + c/x.
WORKED EXAMPLE — IVP

Solve x dy/dx − y = x/(x+1), y(1) = 0.

1Standard form: dy/dx − (1/x)y = 1/(x+1). P = −1/x, ρ = e^(−ln x) = 1/x.
2d/dx(y/x) = 1/[x(x+1)] = 1/x − 1/(x+1) ⟹ y/x = ln x − ln(x+1) + C.
3At (1,0): C = ln 2 ⟹ y = x·ln[2x/(x+1)].
Useful facts: e^(ln P) = P, ∫dx = x, ∫dy = y. Always reduce dy/dx to coefficient 1 before reading off P(x).
Lesson 7.6 — Applications of ODEs▶
WORKED EXAMPLE — population (linear ODE)

dR/dt = 0.07 − 0.1R, R(0) = 7000. Find R after 10 years.

1dR/dt + 0.1R = 0.07; ρ = e^(0.1t). d/dt(R e^(0.1t)) = 0.07 e^(0.1t).
2R = 0.7 + C/e^(0.1t); R(0)=7000 ⟹ C = 6999.3.
3At t = 10: R ≈ 2576 rats.
WORKED EXAMPLE — falling object

dv/dt = g − 4v, v(0) = 0. Show v = (g/4)(1 − e^(−4t)) and find v as t → ∞.

1dv/dt + 4v = g; ρ = e^(4t). d/dt(v e^(4t)) = g e^(4t) ⟹ v e^(4t) = (g/4)e^(4t) + C.
2v(0)=0 ⟹ C = −g/4, so v = (g/4)(1 − e^(−4t)).
3As t → ∞: e^(−4t) → 0, so v → g/4 (terminal velocity).
Newton's cooling worked example from the slides: dθ/dt = −k(θ − 18) with data θ=70 at t=0, θ=50 at t=5 gives k = 0.058; the object reaches θ = 40° at t ≈ 14.83 minutes.

Numerical Methods

Lesson 8.1 — Locating roots: graphs, tables & IVT▶

To solve f(x) = 0 when the exact (analytical) solution is hard, we use numerical methods. First step: locate a small interval that contains a root.

Graphical method

The real zeros of f(x) are where y = f(x) crosses the x-axis. If y = f(x) is awkward to draw, rearrange f(x) = 0 into p(x) = q(x) and find the intersection of the two easier curves.

Example: x⁵ − x − 1 = 0 ⟹ x⁵ = x + 1. Plot p = x⁵ and q = x + 1; they cross at x* ∈ (1, 2). Similarly e^(−x) − x = 0 ⟹ e^(−x) = x gives a root ≈ 0.6.

Table of values & Intermediate Value Theorem (IVT)

If f is continuous on [a,b] and f(a)·f(b) < 0 (opposite signs), then a root lies in (a, b).

WORKED EXAMPLE — table

f(x) = x⁵ − x − 1 on [0,2], step h = 0.5.

1f: −1 (x=0), −1.47 (0.5), −1 (1.0), 5 (1.5), 29 (2.0).
2Sign change between 1.0 and 1.5 ⟹ root in (1, 1.5).
WORKED EXAMPLE — IVT

Verify 4 sin x = e^(−x) has a root in (0, 1.5). Let f(x) = e^(−x) − 4 sin x (x in radians).

1f(0)·f(1.5) = (1)(−3.7668) = −3.7668 < 0 ⟹ root exists in (0, 1.5).
Lesson 8.2 — Fixed-point iteration▶

Rewrite f(x) = 0 as x = g(x), then iterate xᵢ₊₁ = g(xᵢ). The iteration converges only if |g′(x)| < 1 near the root — otherwise it diverges. Stop when |xᵢ − xᵢ₋₁| < ε.

WORKED EXAMPLE — choosing a suitable g

x³ + x² − 1 = 0 has a root in (0,1). Three rearrangements (test at x₀ = 0.75):

ig = 1/√(x+1): |g′(0.75)| = 0.216 < 1 ✓ suitable.
iig = 1/x − x²: |g′(0.75)| = 3.28 > 1 ✗ diverges.
iiig = √(1 − x³): |g′(0.75)| = 1.48 > 1 ✗ diverges.
→Iterate (i): x₁ = 0.7559, x₂ = 0.7546, |x₂−x₁| = 0.0013 < 0.005 ⟹ root ≈ 0.755 (3 d.p.).
WORKED EXAMPLE

x³ − 2x − 5 = 0, root in (2,3). Suitable form x = (2x + 5)^(1/3) (|g′(2)| = 0.154 < 1).

1x₀=2 → 2.08008 → 2.09235 → 2.09422 → 2.09450 → 2.09454.
2|x₄−x₃| = 0.00028 < 0.0005 ⟹ root ≈ 2.0945 (4 d.p.).
ε = 5 × 10⁻ᵐ where m = number of decimal places required.
Lesson 8.3 — Newton–Raphson method▶

A special, faster case using the tangent line. Provided f′(xᵢ) ≠ 0:

xᵢ₊₁ = xᵢ − f(xᵢ)/f′(xᵢ)
WORKED EXAMPLE

x³ = 6x − 4, i.e. f(x) = x³ − 6x + 4, f′(x) = 3x² − 6, x₀ = 1 (3 s.f.).

1x₁ = 1 − (−1)/(−3) = 0.667.
2x₂ = 0.7302, x₃ = 0.7320 (|x₃−x₂| = 0.0018 < 0.005).
3root ≈ 0.732 (3 s.f.).
WORKED EXAMPLE — transcendental

e^x − 2/x = 0, f′(x) = e^x + 2/x², x₀ = 0.8, stop when |f(xᵢ)| ≤ 10⁻⁴.

1x₁ = 0.8 − (−0.2745)/(5.3505) = 0.8513; |f(x₁)| = 0.0066.
2x₂ = 0.8526; |f(x₂)| = 0.00003 < 10⁻⁴ ⟹ root ≈ 0.8526.
Newton–Raphson usually converges in fewer steps than fixed-point iteration, but needs f′(x).
Lesson 8.4 — Numerical integration: Trapezoidal rule▶

To approximate ∫ₐᵇ f(x) dx (the area under the curve) when f can't be integrated, split [a,b] into N strips of width h = (b−a)/N and join the tops with straight lines (trapezoids).

∫ₐᵇ f(x) dx ≈ (h/2)[f₀ + f_N + 2(f₁ + f₂ + … + f_{N−1})]
Counting: 5 ordinates = 4 strips; 6 subintervals = 6 strips. N = number of strips, h = width.
WORKED EXAMPLE — from a data table

x: 1.2,1.3,1.4,1.5,1.6,1.7; f: 2.2,2.1,4.7,6.2,5.5,5.0. N = 5, h = 0.1.

1A ≈ (0.1/2)[2.2 + 5.0 + 2(2.1+4.7+6.2+5.5)] = 2.21.
WORKED EXAMPLE — refining h

∫₁^1.6 f dx: with h = 0.2 (N=3) ⟹ 0.5504; with h = 0.1 (N=6) ⟹ 0.5487 (smaller h, more accurate).

WORKED EXAMPLE — from a function

∫₀⁸ √(1+x²) dx, N = 4, h = 2. f: 1, √5, √17, √37, √65.

1≈ (2/2)[1 + √65 + 2(√5+√17+√37)] = 33.9461.
Lesson 8.5 — Numerical integration: Simpson's rule▶

Fits parabolas instead of straight lines — more accurate than the trapezoidal rule. N must be EVEN.

∫ₐᵇ f(x) dx ≈ (h/3)[f₀ + f_N + 4(odd-index f) + 2(even-index f)]
Memory aid: (h/3)[FIRST + LAST + 4(sum of ODDs) + 2(sum of EVENs)].
WORKED EXAMPLE — data table

x: 2 → 3.2 step 0.2 (N=6); f: 1, 0.5, 0.333, 0.25, 0.2, 0.167, 0.143.

1≈ (0.2/3)[1 + 0.143 + 4(0.5+0.25+0.167) + 2(0.333+0.2)] = 0.3919.
WORKED EXAMPLE — function, 8 strips

∫₀² 1/(x²+x+1) dx, N = 8, h = 0.25.

1FIRST+LAST = 1.1429; sum of ODDs = 1.6286; sum of EVENs = 1.1152.
2≈ (0.25/3)[1.1429 + 4(1.6286) + 2(1.1152)] = 0.8240.
Trapezoidal needs no parity condition; Simpson's needs N even but is more accurate for the same h.
